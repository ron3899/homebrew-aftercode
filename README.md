# Homebrew tap for Aftercode

Install the [Aftercode](https://github.com/ron3899/aftercode) CLI:

```bash
brew install ron3899/aftercode/aftercode
```

That's shorthand for:

```bash
brew tap ron3899/aftercode
brew install aftercode
```

The formula builds from source; Homebrew installs Rust automatically if needed.

Upgrade with `brew upgrade aftercode`.
